# Ascii Donut 🍩

A mesmerizing terminal-based ASCII art donut, rendered in real-time with 3D rotation and lighting effects. Written in pure Rust.

## Overview

This project renders a spinning 3D torus (donut shape) using only ASCII characters in your terminal. It uses mathematical projections, Z-buffering for depth testing, and dot-product-based lighting to create a smooth, animated visualization.

The rendering uses characters from darkest to brightest: `.,-~:;=!*#$@`

## Features

- **Real-time 3D rendering** in the terminal
- **Z-buffer depth tracking** for correct occlusion
- **Lighting calculations** using surface normal dot products
- **Smooth animation** with continuous rotation on two axes
- **Pure Rust** — zero dependencies

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (edition 2024 or later)
- A terminal with ANSI escape code support

## Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/ascii-donut.git
cd ascii-donut
```

### Build

```bash
cargo build --release
```

### Run

```bash
cargo run --release
```

Press `Ctrl+C` to stop the animation.

## How It Works

1. **Torus Parametrization**: The donut is defined by two radii — a tube radius (`r1`) and a major radius (`r2`). Points are generated by iterating over two angles (`theta` and `phi`).

2. **3D Rotation**: Two rotation angles (`a` and `b`) increment each frame, producing continuous spinning motion around the X and Z axes.

3. **Perspective Projection**: 3D coordinates are projected onto the 2D terminal using perspective division (`1/z`), scaled by a constant factor for visual appeal.

4. **Z-Buffering**: A depth buffer ensures only the closest surface point is rendered at each pixel.

5. **Lighting**: A luminance value is computed via the dot product of the surface normal and a light direction vector. This determines which ASCII character is used for each pixel.

## Configuration

You can tweak the following parameters in `src/main.rs`:

| Parameter   | Default | Description                  |
|-------------|---------|------------------------------|
| `width`     | 80      | Terminal width (columns)     |
| `height`    | 40      | Terminal height (rows)       |
| `r1`        | 1.0     | Tube radius                  |
| `r2`        | 2.0     | Donut (major) radius         |
| `k1`        | 35.0    | Projection scale factor      |
| `a` step    | 0.04    | X-axis rotation speed        |
| `b` step    | 0.02    | Z-axis rotation speed        |
| `phi` step  | 0.02    | Torus angle step (detail)    |
| `theta` step| 0.07    | Torus angle step (detail)    |
| sleep       | 30ms    | Frame delay                  |
