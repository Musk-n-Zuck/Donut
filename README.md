# Ascii Donut 🍩

A mesmerizing terminal-based ASCII donut renderer written in Rust. Watch as a spinning 3D torus comes to life using nothing but ASCII characters, mathematical projections, and a bit of trigonometry.

## Overview

This project renders a rotating 3D torus (donut shape) directly in your terminal using ASCII characters. It uses:

- **3D rotation math** — Applies rotation matrices to spin the donut on both the X and Z axes
- **Perspective projection** — Maps 3D coordinates to 2D terminal space with proper depth scaling
- **Z-buffering** — Ensures correct depth rendering so closer pixels overwrite farther ones
- **Luminance-based shading** — Uses dot product lighting to select characters from a palette (`.,-~:;=!*#$@`) based on surface brightness

## How It Works

1. **Torus Parametrization** — The donut is defined by two radii: a tube radius (`r1`) and a major radius (`r2`). Points on the surface are generated by iterating over two angles (`theta` and `phi`).
2. **3D Rotation** — Each point is rotated using rotation matrices around the X and Z axes, controlled by angles `a` and `b`.
3. **Perspective Projection** — 3D coordinates are projected to 2D using inverse depth (`1/z`) for perspective.
4. **Depth Testing** — A z-buffer tracks the closest rendered character at each screen position.
5. **Lighting** — A luminance value is calculated per pixel using a dot product with a simulated light direction. This determines which ASCII character to render, from dim (`.`) to bright (`@`).

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (Edition 2024 or later)

## Building

```bash
cargo build --release
```

## Running

```bash
cargo run --release
```

Press `Ctrl+C` to exit the animation.

## Configuration

You can tweak the following parameters in `src/main.rs`:

| Parameter      | Default | Description                          |
|----------------|---------|--------------------------------------|
| `width`        | 80      | Terminal width in characters         |
| `height`       | 40      | Terminal height in characters        |
| `r1`           | 1.0     | Tube radius of the torus             |
| `r2`           | 2.0     | Major radius of the donut            |
| `k1`           | 35.0    | Projection scale factor              |
| `a` increment  | 0.04    | X-axis rotation speed                |
| `b` increment  | 0.02    | Z-axis rotation speed                |
| Frame delay    | 30ms    | Sleep duration between frames        |
